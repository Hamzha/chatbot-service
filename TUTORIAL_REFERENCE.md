# Production-Ready RAG AI Agent Tutorial Reference

This project follows the tutorial:

**How to Build a Production-Ready RAG AI Agent in Python (Step-by-Step)**  
<https://www.youtube.com/watch?v=AUQJ9eeP-Ls>

## Followed Tutorial Description

This tutorial walks through building a Python RAG (Retrieval-Augmented Generation) app and preparing it for production use, including:

- Orchestration and observability with Inngest
- FastAPI backend setup
- Local Qdrant vector database setup
- PDF loading and chunking with LlamaIndex
- Embeddings and LLM calls with OpenAI
- Streamlit frontend integration
- Production concerns such as retries, throttling, rate limiting, and concurrency

## Tools and Stack Covered

- Python
- FastAPI
- Inngest (workflow orchestration and function execution)
- Qdrant (vector database)
- LlamaIndex (PDF parsing/chunking)
- OpenAI API (embeddings + model responses)
- Streamlit (simple UI)
