# Business Platform Documentation

This folder contains commercialization and implementation docs for turning the chatbot system into a business-ready product.
The plans are now separated into dedicated subfolders for clarity.

## Folder Structure

### inventory-chatbot-plan

- `01-product-vision.md`: Product concept, target users, and value proposition.
- `02-business-owner-inventory-chatbot.md`: Core plan for inventory-fed AI chatbot on WhatsApp and website.
- `03-implementation-roadmap.md`: Technical rollout plan, milestones, and delivery phases.
- `04-pricing-packages.md`: Monetization structure and package recommendations.

### voice-call-center-plan

- `05-voice-call-center-automation-plan.md`: Separate plan for voice-based call center and customer support automation.
